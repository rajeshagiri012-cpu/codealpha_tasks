# CodeAlpha Power BI Internship Projects

## Overview

This repository contains Power BI dashboards developed as part of my Power BI Internship at CodeAlpha. These projects focus on data visualization, business intelligence, and analytical reporting using Power BI.

---

# Project 1: Financial Analysis Dashboard

## Objective

To analyze financial performance and provide insights into revenue, profit, assets, liabilities, and cash flow.

## Key Metrics

* Total Revenue
* Net Profit
* Total Assets
* Total Liabilities
* Cash Inflow
* Cash Outflow

## Features

* Revenue Trend Analysis
* Profitability Analysis
* Forecast Trend Analysis
* Balance Sheet Overview
* Expense Breakdown
* Interactive Filters and Slicers

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling

## Insights

* Identified revenue growth patterns.
* Analyzed profit margins across periods.
* Evaluated financial health using asset and liability metrics.
* Forecasted future revenue trends.

# Project 2: HR Analytics Dashboard

## Objective

To analyze employee data and identify workforce trends, employee attrition, and department performance.

## Key Metrics

* Total Employees
* Attrition Count
* Attrition Rate
* Average Salary
* Average Performance Rating

## Features

* Employee Attrition Analysis
* Department-wise Employee Distribution
* Gender Analysis
* Salary Analysis
* Performance Analysis
* Interactive Dashboard Filters

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling

## Insights

* Identified departments with higher attrition rates.
* Analyzed workforce demographics.
* Evaluated salary distribution across employees.
* Assessed employee performance trends.
